2.Bootstrap Structure and Files

Bootstrap Directory Structure and Purpose

1. CSS Folder

This folder contains the CSS files that define how Bootstrap components look. By including bootstrap.min.css, your web page will have Bootstrap's styling applied, such as responsive grids, buttons, forms, colors, and typography.

2. JS Folder

The JS folder contains JavaScript files that add interactivity to Bootstrap components. For example, it enables modals (pop-up windows), dropdown menus, carousels, tooltips, and more. We include bootstrap.bundle.min.js because it bundles Bootstrap's JavaScript and Popper.js (a library for positioning tooltips and popovers) into a single file for convenience.

3. Icons Folder

This folder holds Bootstrap Icons, which are scalable vector icons designed to work with Bootstrap. They enhance the user interface by providing commonly used symbols (like arrows, check marks, and social media icons). Using SVG icons from this folder helps keep your site lightweight and visually consistent.
